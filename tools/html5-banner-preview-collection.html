How to use these preview pages

Each preview page focuses on one size. Load your banner, then:

  1. Verify that the ad fits perfectly inside the preview frame.
  2. Test your click-through URL.
  3. Check the animation loop – usually 15–30 seconds is enough.
  4. Try different browsers to be sure everything behaves the same.
