Before you send the banner to a network

  1. Check that the click URL works and opens in a new tab.
  2. Verify that the banner stays within the correct size (no overflow).
  3. Watch CPU usage for a few seconds – no excessive looping animations.
  4. Reload several times to make sure there are no random errors.
